# Securitisation Risk Assessment Dashboard

## Overview

This project is a **Power BI-based Securitisation Risk Assessment Dashboard** developed using simulated banking loan portfolio data. It provides insights into portfolio performance, credit risk, Expected Credit Loss (ECL), delinquency trends, and investor reporting through interactive dashboards.

The project applies **IFRS 9 credit risk concepts** and demonstrates data modeling, advanced DAX calculations, and business intelligence techniques for financial risk analysis.

---

## Features

* Portfolio Performance Dashboard
* IFRS 9 Expected Credit Loss (ECL) Analysis
* Days Past Due (DPD) Analysis
* Static Pool (Vintage) Analysis
* Dynamic Loss Analysis
* Stress Testing & Scenario Analysis
* Investor Reporting Dashboard
* Interactive Power BI Visualizations
